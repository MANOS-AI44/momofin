# Dossier des APK auto-hébergés

Placez vos APK compilés ici (filenames exacts) :

- `MoMoSMS-debug.apk`
- `MoMoFin-debug.apk`

Dès que ces fichiers sont présents, la page `/telecharger` les sert directement plutôt qu'en allant chercher la release GitHub. Pratique si vous voulez héberger les APK avec votre backend Railway sans passer par GitHub.
